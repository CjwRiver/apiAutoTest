## 1.1. 融合API V3.0.1 接口说明
- 接口基准地址：`http://meetingpre.125339.ebupt.net
- 除了“获取Token验证码”、“获取Token”、“Token校验”接口，请求融合API的其他接口时，需要将已获取的Token携带在HTTP 头部，如下：**

    | *消息头（Header）* | *是否可选*                | *说明*                                              |
    | -------- | --------------------- | --------------------------------------------------- |
    | Content-Type     | 必选           | application/json; charset=utf-8                                            |
    | X-ACCESS-TOKEN   | 必选           | 调用获取Token接口拿到的token值                                            |
    ------
- **meetingpre中的token值，在config文件中中已写死**
- **需要授权的 API ，必须在请求头中使用 `Authorization` 字段提供 `token` 令牌**
- **使用 HTTP Status Code 标识状态+自定义状态码，以实际为准**
- **数据返回格式统一使用 JSON**

### 1.1.1. 支持的请求方法

- GET（SELECT）：从服务器取出资源（一项或多项）。
- POST（CREATE）：在服务器新建一个资源。
- PUT（UPDATE）：在服务器更新资源（客户端提供改变后的完整资源）。
- PATCH（UPDATE）：在服务器更新资源（客户端提供改变的属性）。
- DELETE（DELETE）：从服务器删除资源。
- HEAD：获取资源的元数据。
- OPTIONS：获取信息，关于资源的哪些属性是客户端可以改变的。

### 1.1.2. 通用返回状态说明

| *状态码* | *含义*                | *说明*                                              |

## 1.2. 会议模块

### 1.2.1. 查询会议列表

- 请求路径：getMeetingList
- 请求方法：get
- 请求参数

    | 参数名   | 必须 | 数据类型     | 默认值 | 备注     |
    | --------      | -------- | -------- | -------- | -------- |
    | meeting_filter | 否   | String  |null|会议状态，（开始R、已结束E、预约W、收藏C），纯软终端会议没有收藏类型的搜索。|
    | meeting_operate_type | 是     | int |2|操作的会议类别，0：公开会议；1：保密会议；2：纯软终端会议|
    | meeting_theme | 否     | String |无|要搜索的关键字，按会议主题模糊搜索。会议类型为纯软终端会议，不支持此字段的搜索。|
    | page | 否     | int |1|当前页码（默认1）|
    | page_size | 否     | int |8|每页显示的条数（默认8）|
    | start_date | 否     | String |无|开始时间格式: yyyy-MM-dd HH:mm:ss，当会议类型为2：纯软终端会议时该字段必填|
    | end_date | 否     | String |无|结束时间格式: yyyy-MM-dd HH:mm:ss，当会议类型为2：纯软终端会议时该字段必填|

- 响应参数

    | 参数名   | 必须    | 数据类型            |默认值|描述|
    | -------- | ----------- | --------------- | ----------- | --------------- |
    | code       | 是     |      int           |null       |返回码|
    | msg      | 是 |         String        |null|对返回码的描述|
    | data | 否      |   page,第几页total，查询结果总数page_size，每页返回几条page_count，总共分页多少页list：<MeetingSimpleDto>[]，会议数据列表
                  |||
